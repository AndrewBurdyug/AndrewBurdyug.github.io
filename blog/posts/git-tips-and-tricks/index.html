Git Tips and Tricks

September 2, 2013   

git

Push

Push the local new branch into the new remote branch:

git push <remote-name> <local-branch-name>:<remote-branch-name>

Delete remote git branch:

git push origin --delete branch_name

Pull & fetch

Delete local stale git branches which were alreay deleted on the remote server:

git fetch -p

Fetch the new git branch and checkout into it:

git fetch origin <remote-branch-name>:<local-branch-name>
git checkout <local-branch-name>

or the shortest variant:

git checkout -b local_branch_name origin/remote_branch_name

Add sub module:

git submodule add -b branch_name git@gitlab.com:repo_group/repo_name.git project_folder/folder_name

Config

Custom format for CHANGELOG:

git log --format="%ad %h %s" --date="short"

printed data will looks like:

2013-01-15 1ff347c Fix: export of data items (Bug #116)
2013-01-15 104c3e6 Change: /user returns all timestamps in UTC (Task #17)
2013-01-09 e6457ef Fix: systemd service is freeze after restart (Bug #14)
2013-01-09 a1515b5 Update: docs (Task #2)

Custom colors in diff mode (add below parameters into your ~/.gitconfig):

[color "diff"]
        meta = white bold
        frag = magenta bold
        old = red
        new = green

Resolve remote git branch name automatically:

git config push.default current

Changing the repo URL to a new one:

git remote set-url origin git://new.url.here

Added git submodules:

git submodule add -b master git@gitlab.com:vehicle/wheel.git vehicle/wheel

Search

Search for a string/regex across all branches:

git grep "string/regexp" $(git rev-list --all)
